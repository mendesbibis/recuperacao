Os principais comandos par se criar um site

Neste site estarei falando de dois dos principais comandos para a criação de um site


HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Defina o significado e a estrutura do conteúdo da página da web. Outras tecnologias além do HTML são frequentemente usadas para descrever a aparência/apresentação (CSS) ou funcionalidade/comportamento (JavaScript) de páginas da web.


CSS

É uma linguagem de marcação amplamente utilizada com HTML ou XHTML e representa uma variedade de possibilidades de formatação. CSS ajuda você a editar, alinhar, remover e manipular o espaço entre os elementos da página.
